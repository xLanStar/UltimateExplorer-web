# Ultimate Explorer - Web

這個只是 Ultimate Explorer 的前端網頁，並不包含後端伺服器。需要將輸出的網頁放到 Ultimate Explorer 的伺服器中指定的網頁資料夾，才能夠運行。

## 需要的 Node.js 套件

- @vitejs/plugin-vue
- vite
- vue-axios

---

## 建置

`npm run build`

### 改變建置資料夾位置

在 vite.config.js 修改 outDir 路徑
